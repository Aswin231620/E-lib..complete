# **App Name**: OpenShelf

## Core Features:

- Book Browsing: Allows users to browse books by category and subject.
- Book Search: Enables users to search for books by title, author, and category using keywords.
- Book Details: Displays detailed information about each book, including title, author, publication year, category, and a short description.
- User Authentication: Provides user registration, login, and guest access functionalities.
- Content Management: Admin panel for uploading books, editing book details, and removing content.
- Digital Storage: Organize the digital books in appropriate categories
- Resource Management: Provide all available ebooks with simple navigation to find what they need quickly.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for a sense of trust and reliability.
- Background color: Light blue (#E3F2FD), very lightly saturated, for a clean and non-distracting backdrop.
- Accent color: Soft orange (#FFAB40), analogous to blue but offering warmth, to highlight interactive elements and call-to-actions.
- Font pairing: 'Literata' (serif) for body text, lending a literary feel, paired with 'PT Sans' (sans-serif) for headlines, for modern clarity.
- Simple, clear icons representing book categories and user actions.
- Clean and intuitive layout with clear navigation for easy browsing.
- Subtle transitions and feedback animations for a smooth user experience.
